# 🍽 Сімейне меню

Сімейний сайт для планування меню на тиждень — сніданок, обід і вечеря на кожен день.  
Список продуктів генерується автоматично зі запланованих страв.

## Функції

- 📅 **Планувальник на тиждень** — сітка 7 днів × 3 прийоми їжі
- ✏️ **Редагування страв** — назва, нотатки/рецепт, інгредієнти
- 🛒 **Список покупок** — 5 категорій, автоматична генерація з меню
- ✅ **Відмітка куплених** — чекбокси та закреслення
- 💾 **Автозбереження** — дані зберігаються у `localStorage` браузера
- 📱 **Адаптивний дизайн** — працює на мобільних пристроях

## Структура файлів

```
family-menu/
├── index.html   # розмітка сторінки
├── style.css    # стилі (нейтральна палітра)
├── app.js       # вся логіка застосунку
└── README.md
```

## Швидкий старт

1. Скопіюйте репозиторій або завантажте файли
2. Відкрийте `index.html` у браузері — готово!

Жодних залежностей і збірки не потрібно.

## Деплой на GitHub Pages

```bash
git init
git add .
git commit -m "init: сімейне меню"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГІН/family-menu.git
git push -u origin main
```

Потім у налаштуваннях репозиторію → **Pages** → Source: `main / (root)`.  
Сайт буде доступний за адресою `https://ВАШ_ЛОГІН.github.io/family-menu/`
